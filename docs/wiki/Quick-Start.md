# Быстрый старт

Эта инструкция поможет сделать первую бронь за 2-3 минуты.

## 1) Откройте главную страницу

На главной странице нажмите кнопку **Найти столик**.

## 2) Перейдите в каталог ресторанов

На странице ресторанов:

- используйте строку поиска;
- выберите фильтры: кухня, цена, рейтинг;
- откройте понравившийся ресторан.

## 3) Проверьте информацию о ресторане

В карточке ресторана можно посмотреть:

- фотографии;
- описание и тип кухни;
- адрес, телефон, сайт;
- часы работы;
- меню по категориям.

## 4) Заполните форму бронирования

В блоке **Забронировать столик** укажите:

- дату;
- время;
- количество гостей;
- имя;
- email;
- телефон;
- особые пожелания (необязательно).

## 5) Отправьте бронь

Нажмите **Забронировать столик**.  
Если обязательные поля заполнены корректно, появится сообщение об успешной отправке.

## Советы для первого бронирования

- Выбирайте дату не в прошлом (форма блокирует прошедшие даты).
- Для популярных ресторанов бронируйте заранее.
- Указывайте корректный номер телефона для связи.
