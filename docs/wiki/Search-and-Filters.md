# Поиск и фильтры ресторанов

Страница **Рестораны** помогает быстро выбрать подходящее место.

## Строка поиска

Поиск работает по:

- названию ресторана;
- типу кухни;
- локации (району).

Пример: введите `Итальянская` или `Центр`.

## Фильтр "Кухня"

Позволяет оставить в списке только один тип кухни:

- французская;
- японская;
- итальянская;
- и другие доступные варианты.

## Фильтр "Цена"

Используйте диапазон:

- `₽` — бюджетно;
- `₽₽` — средний сегмент;
- `₽₽₽` — премиум.

## Фильтр "Рейтинг"

Доступные пороги:

- `4.0+`
- `4.5+`

## Сброс фильтров

Если подходящих вариантов нет, нажмите **Сбросить фильтры**.  
Это вернет исходные значения поиска и фильтрации.

## Рекомендации

- Сначала задайте кухню и район, потом уточняйте рейтинг.
- Для быстрого выбора используйте короткие запросы по одному критерию.
