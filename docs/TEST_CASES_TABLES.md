## Раздел 1. Авторизация (TC-01 … TC-10)

### TC-01

| Поле | Значение |
|------|----------|
| **Название** | Детерминированное хеширование пароля |
| **Функция** | Безопасное хранение пароля (`hashPassword`) |
| **Действие** | Дважды вычислить хеш одного и того же пароля |
| **Ожидаемый результат** | Оба хеша совпадают; длина хеша — 64 символа (SHA-256) |
| **Результат теста** | ● пройден |
| **Предусловие** | Доступен Web Crypto API в тестовой среде |
| **Шаги теста** | 1. Вызвать `hashPassword("secret123")`<br>2. Повторно вызвать `hashPassword("secret123")`<br>3. Сравнить результаты |
