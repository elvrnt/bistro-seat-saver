# 🍽️ Restaurant Table Booking App

Веб-приложение для онлайн-бронирования столиков в ресторанах.  
Позволяет пользователям просматривать рестораны, изучать меню, часы работы и бронировать столики в несколько кликов.

---

## 📋 О проекте

Современное одностраничное приложение (SPA), разработанное с акцентом на удобство пользователя, адаптивный дизайн и производительность.

### Основной функционал:

- Просмотр списка ресторанов с фото, рейтингом, типом кухни и ценовым диапазоном
- Фильтрация и поиск ресторанов
- Детальная страница ресторана:
  - Галерея изображений
  - Контакты (адрес, телефон, сайт)
  - Часы работы по дням недели
  - Меню с категориями (закуски, основные блюда, десерты, напитки)
- Форма бронирования столика:
  - Выбор даты, времени, количества гостей
  - Контактные данные (имя, email, телефон)
  - Пожелания к бронированию
  - Валидация всех полей
- Адаптивность под ПК, планшеты и мобильные устройства

## 🚀 Быстрый старт

### Требования

- Node.js (версия 18 или выше)
- npm или yarn

### Установка и запуск

```bash
# Клонирование репозитория
git clone https://github.com/your-username/restaurant-booking.git

# Переход в папку проекта
cd restaurant-booking

# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev
